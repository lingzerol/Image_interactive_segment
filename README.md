# Image_interactive_segment
交互式图像分割。完全自动的图像分割技术如今尚未成熟，无法完成功的将物体和背景有效额分割开。因此，本项目将会采用交互方式，对图像的物体和背景进行分割。相比与以前算法，该项目改进了超像素算法和图像区域融合算法，对其速度和融合效果有暨大的提高。

功能
----

程序将会要求使用者输入照片，并对照片进行标记。程序将会根据部分标记进行区域的分割，并最终形成物体和背景两个区域块。
